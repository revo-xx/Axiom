# Axiom

A modern, professional, and secure game cheat loader built with .NET 8 WPF, featuring a matching landing page.

## Features

- **Modern UI/UX**: Sleek dark theme with smooth fade-in animations.
- **Modular Design**: Clean C# code structure using UserControls for easy maintenance.
- **Dynamic Theming**: Real-time theme color switching (Blue, Red, Green).
- **Secure Profile**: HWID-based user identification and subscription tracking.
- **Game Injector**: Support for multiple games like CS2, Valorant, and Minecraft.
- **Responsive Website**: A professional landing page to showcase your brand.

## Tech Stack

- **Loader**: .NET 8.0, WPF, XAML.
- **Website**: HTML5, CSS3, FontAwesome.
- **Icons**: Windows Segoe MDL2 Assets (No external dependencies).

## Project Structure

- `AxiomLoader/`: The main WPF application source code.
- `AxiomWebsite/`: The brand's landing page files.

## Getting Started

1. Clone the repository.
2. Open the solution in Visual Studio 2022 or VS Code.
3. Navigate to `AxiomLoader` and run `dotnet run`.
4. Open `AxiomWebsite/index.html` to view the landing page.
